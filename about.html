About me

Short version:

I'm a self-taught software developer, specializing in front-end developmemnt . I started coding from a young age and since then I've built lots of projects. What I like the most in coding is the process to solve/create something.

Long version: (for the curious ones)

My background:

I've been interacting with computers from a young age, mainly playing games, but thats what brought me into the programming world. I wanted to make games like the ones I played and so from there it all started to unfoald.

At 16 I beggin learning Python from a series of lectures that a friend of mine gave me in a pen drive.

However, at that time I had school, and the time was little, so I spent most of my free time learning to code.

Time passed... and I've built projects in Python - a web scraper, a card game, a student grades program, etc...

At 18 I was into the Microsoft .NET stack, learnt c# and the basics of back-end development and http requests.

Finally, today I can code full fledged web apps in JavaScript, mainly using React and Node.js.

Why the self-taught route?

As explain before, I've learnt alot on my own, and even in a school enviroment most of the things people learn is on their own. So has time passed I was already doing what I wanted, web development.

Principles as a programmer:

I believe that the best way to get better is to try building something you can't do right now.

My coding workflow is always based on a previous TODO list I've made.
