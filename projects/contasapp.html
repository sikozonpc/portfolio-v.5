Contas App

What it is?

It's an application for registering actions (whatever actions) , which makes it broader. In it's core, all it does is some calculations and then displays the data in a "pretty way" for a none technical user to use.

Why built it?

This was my very first requested application by a client (my parents). They wanted an application to registry their spendings and earns, monthly, and then show it in easy to read way.

Technoligies used

    Front-end

  • React
  • SPA - Single page app

    Back-end

  • Node.js
  • Express API Server

    Database

  • MongoDB, using mLab

    Deployment

  • Heroku

Technical Notes:

This was also my first Fullstack application made with JavaScript. So thats why the UI might not be the best, since I had a lot of work on the back-end also.
