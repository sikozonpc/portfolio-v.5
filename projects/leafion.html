What it Leafion?

It's an application for managing your savings by registering your transactions. I can proudly say that Leafion is the application I took the most time to make, it was a huge learning experience. It was also my first full-stack apllication.

Why built it?

This was my very first requested application by a client (my parents). They wanted an application to registry their spendings and earns, monthly, and then show it in easy to read way. And so I made it available for everyone after many changes.

Technoligies used

    Front-end

  • React
  • Redux

    Back-end

  • Node.js
  • Google's Firebase service
  • Express API Server

    Database

  • MongoDB, using mLab
  • Firebase

    Deployment

  • Heroku

Technical Notes:

This was also my first Fullstack application made with JavaScript. I had many challenges with this projects which made me a better developer.
